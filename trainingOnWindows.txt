This file describes how to use opencv in python on Windows.


Download pre-compiled opencv version:

> go to: https://sourceforge.net/projects/opencvlibrary/files/
> find and open latest opencv 3.4 version 
> download windows .exe version.
> after download click on .exe file
> in showd dialog window select folder to which you want to extract precompiled opencv ideally to project folder
  (let's say the opencv was downloaded and unwrapped in C:\Users\UserX\Desktop\PIA_Project)



For training  commands see: training.txt.
